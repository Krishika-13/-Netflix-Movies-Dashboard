This project is an end-to-end analytics dashboard built to explore movie trends and viewer preferences using a Netflix-style dataset. The goal was to convert raw data (movies, ratings, tags, and genome information) into clear, actionable insights through SQL-based analysis and interactive dashboards.

Key Highlights
Cleaned and prepared movie, rating, and tag datasets for analysis.
Used SQL queries to identify top genres, highest-rated categories, popular movies, and trend patterns.
Built an interactive dashboard in Power BIwith filters (genre, year, rating, etc.) for easy exploration.
Focused on creating a stakeholder-friendly dashboard with clear KPIs and visual storytelling.

Insights Covered
Most popular genres and their average ratings
Top-rated movies and categories
Genre trends over time
Relationship between ratings and popularity/engagement

Tools Used
MySQL (data querying & transformation)
Power BI (dashboard & visualization)
Excel (basic preprocessing)
